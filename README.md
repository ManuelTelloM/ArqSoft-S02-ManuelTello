## Actividad 2  -   Juego ahorcado

Este es un readme del juego Ahorcado.

# De que trata.
Esta actividad fue realizada en el salon de clases con los códigos que el profesor nos asigno, el objetivo del juego es adivinar la palabra en la cantidad de intentos asignada.

# Que se hizo

Se creo en el visual Studio community un nuevo proyecto en el cual desarrollamos el juego del ahorcado. utilizando el material proporcionado por el profesor, conforme se fue trabajando se le fueron añadiendo más opciones y hasta un menu de seleccion de categorias de las preguntas a adivinar.

# Como funciona.
El objetivo es adivinar la palabra oculta letra por letra, si el jugador se equivoca en alguna letra, el juego ira creando poco a poco el dibujo de un ahorcado, cabe recalcar que por cada error de letra se le ira agregando una parte del cuerpo al muñeco
Si se completa el dibujo del muñeco, se da por perdido el juego y te revela cual era la palabra oculta.

Para ganar el juego es importante adivianr la palabra correctamente dentro de los intentos disponibles.
