## Actividad 2  -   Juego ahorcado
 Esta actividad se realizó en clases, consiste en un juego de ahorcado que se ejecuta en consola al cual se le agregaron caracteristicas
 como por ejemplo: si el jugador comete 3 errores se le dara una pista de la letra con la que empieza la palabra.
 Cada jugador tiene como máximo 5 intentos.
 Al ejecutar el programa se cargara una lista con las tres posibles categorias las cuales pueden ser POO, .NET y Arquitectura.
