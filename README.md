# README de la viborita

 Este es un readme de la viborita.

## De que trata.
Esta es una actividad realizada durante la clase que consta en el juego de la viborita que tiene que comer sus alimentos para ir creciendo, el punto es que la viborita crezca lo más que se pueda, en dado caso de que choque con alguna de las paredes se termina el juego.

## Que se hizo
Se creo una proyecto en el visual studio comunity, utilizando los códigos que proporciono el profesor durante la clase, se furon implementando nuevas clases y funciones dentro del programa para lograr tener la funcionalidad del juego de viborita.

## Como funciona
El juego consiste en manejar una viborita que tiene que recoger comida y cada vez que se recoge una comida, la viborita va creciendo por lo tanto el juego se va haciendo más complicado.

El juego se pierde si la viborita choca con alguna pared o con su cola.

Para ganar el juego se tiene que lograr sobrevivir el mayor tiempo posible y que la viborita crezca lo más que se pueda.
